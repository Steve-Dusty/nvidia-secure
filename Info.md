  Direct API Access                                                                                                                                                                                        
                                                                                                                                                                                                           
  # Health Facilities                                                                                                                                                                                      
  curl "https://data.sfgov.org/resource/jhsu-2pka.json" > sf_health_facilities.json                                                                                                                        
                                                                                                                                                                                                           
  # Medical Incidents (EMS)                                                                                                                                                                                
  curl "https://data.sfgov.org/resource/nuek-vuh3.json?$limit=50000" > sf_medical_incidents.json                                                                                                           
                                                                                                                                                                                                           
  # Businesses (filter for pharmacies)                                                                                                                                                                     
  curl "https://data.sfgov.org/resource/g8m3-pdis.json?$where=dba_name%20like%20%27%25PHARMACY%25%27" > sf_pharmacies.json

⏺ Here are the specific NVIDIA NIM models used for inference in this project:                                                                                                                              
                                                                                                                                                                                                           
  Visual Inference Models                                                                                                                                                                                  
  ┌───────────────────────┬────────────────────────────┬──────────────────────────────────────────────────┐                                                                                                
  │         Model         │          Endpoint          │                     Purpose                      │                                                                                                
  ├───────────────────────┼────────────────────────────┼──────────────────────────────────────────────────┤                                                                                                
  │ Microsoft Florence-2  │ microsoft/florence-2       │ Scene understanding, action detection (VQA)      │                                                                                                
  ├───────────────────────┼────────────────────────────┼──────────────────────────────────────────────────┤                                                                                                
  │ NVIDIA Grounding DINO │ nvidia/grounding-dino      │ Object/person detection with language grounding  │                                                                                                
  ├───────────────────────┼────────────────────────────┼──────────────────────────────────────────────────┤                                                                                                
  │ Meta SAM2             │ meta/sam2-hiera-large      │ Segment Anything for precise person segmentation │                                                                                                
  ├───────────────────────┼────────────────────────────┼──────────────────────────────────────────────────┤                                                                                                
  │ NVIDIA BodyPose       │ nvidia/bodypose-estimation │ 17-keypoint pose estimation (COCO format)        │                                                                                                
  └───────────────────────┴────────────────────────────┴──────────────────────────────────────────────────┘                                                                                                
  Capabilities:                                                                                                                                                                                            
  - Person detection and tracking                                                                                                                                                                          
  - Action recognition: standing, walking, running, fighting, falling, lying down                                                                                                                          
  - Fall detection via pose keypoints                                                                                                                                                                      
  - Fight detection via erratic movement patterns
